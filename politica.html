Política de privacidade

A Flux e todos os seus colaboradores (“Nós”) leva a privacidade a sério. Por isso, nesta Política de Privacidade (“Política”) explicamos as formas como nós utilizamos, armazenamos, compartilhamos e protegemos os dados pessoais coletados a partir do uso dos nossos serviços e soluções (coletivamente “Serviços”) e da visita aos nossos websites e como os titulares do dados podem exercer os direitos previstos na legislação aplicável.

Os dados pessoais que coletamos dependem do contexto das suas interações conosco. Para facilitar a sua compreensão, explicamos nesta Política como tratamos os dados de (i) clientes (“Clientes”), consumidores de Clientes (“Consumidores”) e de usuários dos nossos websites (“Usuários”), coletivamente (“Titulares” ou “você”).

Políticas de Privacidade específicas podem ser aplicáveis para alguns dos nossos produtos e serviços. Para mais informações sobre nossas práticas de privacidade relacionadas a um produto ou serviço específico, visite a página web deste produto ou serviço.

Quem somos nós +

A Flux! Crédito inteligente para o licitante é uma fintech de Brasília que desde 2020 desenvolve soluções de crédito para empresas que participam de licitações públicas.

Definições importantes +

LGPD: Lei nº 13.709/2018 que regula o tratamento de dados pessoais.

Agentes de Tratamento: Controlador e Operador.

Dados Pessoais: Informações relacionadas à pessoa natural identificada ou identificável.

Tratamento: Toda operação realizada com dados pessoais.

Clientes +

Dados que coletamos

Dados fornecidos diretamente, dados automáticos e dados obtidos de outras fontes.

Como utilizamos esses dados

  • Execução de contratos
  • Segurança
  • Comunicação
  • Conformidade legal
Consumidores +

Esta seção se aplica aos dados pessoais de consumidores dos nossos clientes.

Dados coletados

  • Dados recebidos dos clientes
  • Dados automáticos de navegação
  • Dados de outras fontes
Usuários +

Dados que coletamos

  • Nome, e-mail, telefone
  • CPF e CNPJ
  • Dados de navegação

Como utilizamos estes dados

  • Melhorar os websites
  • Segurança
  • Marketing
Como compartilhamos dados pessoais +
  • Prestadores de serviços
  • Entidades de proteção ao crédito
  • Autoridades públicas
  • Transferência de ativos
  • Com consentimento
Cookies e Tecnologias de Rastreamento +

Utilizamos cookies e tecnologias similares para melhorar a experiência do usuário, personalizar conteúdo e garantir segurança.

Transferência de dados para o exterior +

Dados podem ser transferidos para prestadores de serviços no exterior, respeitando a legislação aplicável.

Interações com sites de terceiros +

Não nos responsabilizamos por políticas de privacidade de sites de terceiros.

Segurança +

Adotamos práticas técnicas e organizacionais para proteger os dados pessoais.

Direitos do titular +
  • Acesso
  • Correção
  • Exclusão
  • Portabilidade
  • Revogação de consentimento
Término do tratamento +

Os dados são tratados enquanto necessários para cumprir obrigações legais ou contratuais.

Alteração na política de privacidade +

A política pode ser alterada a qualquer momento, com aviso aos usuários.
